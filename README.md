# Go Portfolio

A simple, fast, SEO-optimized portfolio site built with Go.

## Usage

### Development

```
make dev
```

### Building the Static Site

```
make build
```

### Serving the Static Site

```
make serve
```

## Docker

Build and run with Docker:

```
docker build -t go-portfolio .
docker run -p 8080:8080 go-portfolio
```
